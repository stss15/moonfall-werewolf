# Third-party notices

Moonfall uses [Trystero](https://github.com/dmotz/trystero) for encrypted peer-to-peer browser connections and [qrcode-generator](https://github.com/kazuhikoarase/qrcode-generator) for the lobby's invite QR codes. Both are available under the MIT License; their dependency notices are recorded by `package-lock.json` and installed through npm.

The automatic narrator plays a pre-recorded voice pack generated by `scripts/generate_voice_pack.py` using one of three free engines chosen at generation time: ElevenLabs (free tier — requires attribution and is non-commercial; used only when a repository owner supplies their own API key), Microsoft Edge neural voices via the `edge-tts` package, or the Apache-2.0 licensed Kokoro open-weights model running locally. When the ElevenLabs engine is used, the optional looping night/day ambience and hero sound stings in `assets/ambience/` are also generated with ElevenLabs sound generation. Narration and ambience audio by elevenlabs.io. When no pack is generated, the browser's built-in Web Speech API voices and the procedural soundscape are used. No paid API is required by any path.

Four short foley transients from Kenney's [Impact Sounds](https://kenney.nl/assets/impact-sounds) pack are layered beneath Moonfall's procedural kill, selection, healing and Hunter effects. Kenney released the pack under [Creative Commons Zero (CC0 1.0)](https://creativecommons.org/publicdomain/zero/1.0/), permitting personal and commercial use without attribution. The remaining sound design and the night/day ambience are generated at runtime with the Web Audio API.

The game’s mechanics are an original software adaptation of classic public social-deduction gameplay. Moonfall’s code, interface copy and fantasy artwork are original to this project and are not official Zygomatic or Asmodee assets.
