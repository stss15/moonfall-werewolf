# Third-party notices

Moonfall uses [Trystero](https://github.com/dmotz/trystero) for encrypted peer-to-peer browser connections and [qrcode-generator](https://github.com/kazuhikoarase/qrcode-generator) for the lobby's invite QR codes. Both are available under the MIT License; their dependency notices are recorded by `package-lock.json` and installed through npm.

The Storyteller voice-over plays a pre-recorded voice pack generated by `scripts/generate_voice_pack.py` using one of three free engines chosen at generation time: ElevenLabs (free tier — requires attribution and is non-commercial; used only when a repository owner supplies their own API key), Microsoft Edge neural voices via the `edge-tts` package, or the Apache-2.0 licensed Kokoro open-weights model running locally. When no pack is generated, the browser's built-in Web Speech API voices are used. All sound effects and ambience are generated procedurally with the Web Audio API. No paid API is required by any path.

The game’s mechanics are an original software adaptation of classic public social-deduction gameplay. Moonfall’s code, interface copy and fantasy artwork are original to this project and are not official Zygomatic or Asmodee assets.
