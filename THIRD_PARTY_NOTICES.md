# Third-party notices

Moonfall uses [Trystero](https://github.com/dmotz/trystero) for encrypted peer-to-peer browser connections and [qrcode-generator](https://github.com/kazuhikoarase/qrcode-generator) for the lobby's invite QR codes. Both are available under the MIT License; their dependency notices are recorded by `package-lock.json` and installed through npm.

The Storyteller voice-over uses the browser's built-in Web Speech API voices, and all sound effects and ambience are generated procedurally with the Web Audio API — no external audio service or paid API is involved.

The game’s mechanics are an original software adaptation of classic public social-deduction gameplay. Moonfall’s code, interface copy and fantasy artwork are original to this project and are not official Zygomatic or Asmodee assets.
